## Objective

Implement `clotho ingest <file>` to read a file from disk and store it as a content blob plus entity row in the workspace.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] `IngestArgs` struct with: `file` (required path), `--type` (meeting/note/transcript/artifact, default note), `--title` (optional, defaults to filename)
- [ ] Reads file content from disk
- [ ] Creates an `EntityRow` in `entities.db` with the appropriate type and title
- [ ] Writes content via `ContentStore`
- [ ] Registers a graph node for the new entity
- [ ] Indexes the content in FTS5 for search
- [ ] Prints entity id and title on success
- [ ] `--json` flag outputs full entity details as JSON
- [ ] Returns error if file not found or workspace not initialized
- [ ] `cargo build` succeeds with no errors
