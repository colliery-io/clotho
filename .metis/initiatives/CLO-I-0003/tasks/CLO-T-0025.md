## Implementation Notes

### Dependencies
Depends on: CLO-T-0021 (EntityStore), CLO-T-0024 (SearchIndex) — needs databases to exist

### Technical Approach
In `src/federation.rs`. Thin wrapper — opens a rusqlite connection, ATTACHes the other databases, runs a query. graphqlite already uses ATTACH internally so the pattern is proven. This is about providing a clean API for clotho's cross-layer queries.
