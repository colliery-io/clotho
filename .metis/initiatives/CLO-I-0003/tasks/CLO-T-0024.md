## Implementation Notes

### Dependencies
Depends on: CLO-T-0019 (scaffold), CLO-T-0021 (EntityStore), CLO-T-0020 (ContentStore)

### Technical Approach
In `src/index.rs`. Uses rusqlite with FTS5 feature. The index is gitignored and can be rebuilt from entities.db + content/ files at any time. Search uses FTS5 MATCH with BM25 ranking.
