## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `report-builder` agent — an autonomous worker that aggregates program data and formats it for a specific audience. Launched by the /report skill with program, time period, audience type, and emphasis parameters.

## Agent Flow

1. Query objectives BELONGS_TO selected program(s)
2. List tasks BELONGS_TO program grouped by state
3. List tasks completed in period
4. List decisions linked to program
5. List active risks and unresolved blockers
6. List artifacts delivered (DELIVERS against objectives)
7. Format for audience:
   - **Boss**: executive summary — outcomes not tasks
   - **Stakeholders**: balanced — progress + risks + timeline
   - **Team**: detailed — task-level breakdown
8. Create Artifact entity with report content
9. Link to program(s) via DELIVERS
10. Present for review
