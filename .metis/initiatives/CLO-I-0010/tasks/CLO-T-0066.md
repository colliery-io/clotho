## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `/period-review` skill — a quarterly/half-year/annual ceremony. Gathers scope from the user, launches the period-compiler agent for deep analysis, then collaborates extensively on the resulting narrative.

## Ceremony Flow

1. Prompt: "What period?" — Q1 / Q2 / H1 / 2025 / custom range
2. Prompt: "Which programs?" — all or specific
3. Prompt: "Focus areas?" — accomplishments / challenges / growth / all
4. Prompt: "Context?" — QBR / self-assessment / performance review / stakeholder update
5. Launch period-compiler agent
6. Present narrative for extensive user collaboration
7. Prompt: "What would you do differently?"
8. Prompt: "What should next period focus on?"
9. Update Reflection entity with user's collaborative input
