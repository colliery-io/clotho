## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `period-compiler` agent — an autonomous deep analysis worker for quarterly+ reviews. Launched by the /period-review skill to aggregate data across programs over a longer time horizon and produce a narrative reflection.

## Agent Flow

1. Query objectives met vs missed across programs in period
2. Query artifacts delivered (DELIVERS)
3. List decisions in period and trace to meetings/programs
4. Analyze decision outcomes: did related tasks complete? did risks resolve?
5. Risk retrospective: risks that have corresponding blockers (materialized) vs risks with no blockers (mitigated)
6. Cross-program pattern identification: consistent progress vs stalled, resource concentration, recurring blocker types
7. Create Reflection entity (quarterly or adhoc period type) with narrative sections:
   - Accomplishments
   - Decisions & Outcomes
   - Challenges
   - Learnings
   - Next Period Focus
8. Link to programs via RELATES_TO
9. Present for user collaboration
