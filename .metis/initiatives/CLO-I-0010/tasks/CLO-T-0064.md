## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `/report` skill — an on-demand ceremony for generating audience-appropriate status reports. Gathers scope from the user (program, time period, audience, emphasis), then launches the report-builder agent.

## Ceremony Flow

1. Prompt: "Which program(s)?" — present list of active programs
2. Prompt: "What time period?" — last week / 2 weeks / month / custom
3. Prompt: "Who is the audience?" — boss / stakeholders / team
4. Prompt: "Anything to highlight or downplay?"
5. Launch report-builder agent with all parameters
6. Present report for review and refinement
