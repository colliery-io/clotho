## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `/weekly-review` skill — an end-of-week ceremony interactive entry point. Gathers the week's data, guides the user through reflection questions, then launches the review-compiler agent to identify patterns and create a Reflection entity.

## Ceremony Flow

1. Auto: list tasks completed this week (state=done, updated_at this week)
2. Auto: list tasks started (state=doing, created this week)
3. Auto: list blocked tasks
4. Auto: list Decisions created this week
5. Auto: list Risks created this week
6. Auto: list Meetings this week
7. Present week summary
8. Prompt: "What went well?"
9. Prompt: "What was harder than expected?"
10. Prompt: "Any programs falling behind? Recurring blockers?"
11. Launch review-compiler agent with all context
12. After agent completes: present reflection for review
13. Prompt: "What should next week focus on?"
