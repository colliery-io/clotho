## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `review-compiler` agent — an autonomous worker for weekly review pattern identification and reflection creation. Launched by the weekly-review skill after data gathering and user reflection prompts.

## Agent Flow

1. Aggregate week's data across all programs
2. Identify patterns:
   - Programs with more blocked than active tasks
   - Workstreams with no completed tasks
   - Blockers affecting multiple tasks
   - Risks open >2 weeks
   - Unanswered questions
3. Create Reflection entity (period=weekly) with structured content
4. Link reflection to relevant programs (RELATES_TO)
5. Present findings: patterns identified, problem areas, and the reflection content
