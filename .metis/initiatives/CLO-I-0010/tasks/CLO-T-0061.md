## Parent Initiative

[[CLO-I-0010]] — Cadence-Driven Agent Ceremonies

## Objective

Create `/daily-brief` skill — a start-of-day ceremony. Skill only, no agent. Queries workspace state via MCP tools and presents a prioritized view of the day ahead.

## Ceremony Flow

1. Prompt user: "Anything new this morning?"
2. Auto: scan inbox for unprocessed files
3. Auto: list tasks where state=blocked + get BLOCKED_BY relations
4. Auto: list tasks with deadlines today/this week
5. Auto: list tasks where state=doing
6. Auto: list open Questions and active Risks
7. Auto: check for pending extraction drafts
8. Synthesize in priority order: inbox alerts → blocked items → due today → in progress → due this week → open questions/risks → stale items from yesterday's debrief
