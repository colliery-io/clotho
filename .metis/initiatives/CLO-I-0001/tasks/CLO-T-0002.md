## Objective

Define all shared types in `src/domain/types.rs`. These are the building blocks used by all traits and entities. All types derive Serialize/Deserialize.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] `EntityId` — newtype wrapping Uuid, with Display, From<Uuid>, new() factory
- [ ] `EntityType` — 15-variant enum (Program, Responsibility, Objective, Workstream, Task, Meeting, Transcript, Note, Reflection, Artifact, Decision, Risk, Blocker, Question, Insight, Person)
- [ ] `Status` — enum: Active, Inactive
- [ ] `TaskState` — enum: Todo, Doing, Blocked, Done
- [ ] `ExtractionStatus` — enum: Draft, Promoted, Discarded
- [ ] `SourceSpan` — struct: transcript_id (EntityId), start (usize), end (usize)
- [ ] `Tag` — newtype wrapping String
- [ ] `PeriodType` — enum: Daily, Weekly, Monthly, Quarterly, Adhoc
- [ ] `Cadence` — struct: frequency (Frequency), cron (Option<String>), label (Option<String>), next_occurrence (Option<DateTime<Utc>>)
- [ ] `Frequency` — enum: Daily, Weekly, Biweekly, Monthly, Quarterly, Yearly, Custom
- [ ] All types derive: Debug, Clone, PartialEq, Serialize, Deserialize
- [ ] `cargo build` succeeds
