## Objective

Implement 4 read-only MCP tools following the Metis `#[mcp_tool]` pattern for searching, querying, reading, and listing entities.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] `clotho_search` tool accepts query (string) and optional limit, delegates to `SearchIndex`
- [ ] `clotho_query` tool accepts cypher (string), delegates to `GraphStore::raw_cypher`
- [ ] `clotho_read_entity` tool accepts entity_id (string), reads from `EntityStore` + `ContentStore`, returns metadata + content
- [ ] `clotho_list_entities` tool accepts optional type/status/state filters, delegates to `EntityStore`
- [ ] All 4 tools use the `#[mcp_tool]` macro with `read_only_hint = true`
- [ ] All tools are registered in `handle_list_tools_request` and dispatched in `handle_call_tool_request`
- [ ] `cargo build` succeeds with no errors
