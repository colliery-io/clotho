## Objective

Implement 4 write MCP tools following the Metis pattern for workspace initialization, content ingestion, note creation, and reflection creation.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] `clotho_init` tool accepts path (string), delegates to `Workspace::init`
- [ ] `clotho_ingest` tool accepts file_path (string) and optional type/title, reads file and creates entity + content + graph + index entries
- [ ] `clotho_create_note` tool accepts title and content (strings), creates a Note entity
- [ ] `clotho_create_reflection` tool accepts period (string) and optional title/program, creates a Reflection entity with template content
- [ ] All tools use `#[mcp_tool]` with `destructive_hint` and `read_only_hint` set appropriately
- [ ] All tools are registered in `handle_list_tools_request` and dispatched in `handle_call_tool_request`
- [ ] `cargo build` succeeds with no errors
