## Objective

Replace the placeholder `Graph` type in traits.rs with `GraphStore`, and update all entity Relatable impls to use the real graph.

## Implementation Notes

### Dependencies
Depends on: CLO-T-0013 (GraphStore), CLO-T-0015 (edge CRUD — needed for Relatable to query edges)

### Technical Approach
The Relatable trait's `relations()` method now calls `graph_store.get_edges_from(self.id())` and maps the results to `Relation` structs. The placeholder `Relation` type in traits.rs can either be kept as a domain type or replaced with `EdgeInfo` from the graph module.
