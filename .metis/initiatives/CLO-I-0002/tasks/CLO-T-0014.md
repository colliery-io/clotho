## Objective

Implement node CRUD operations on GraphStore for registering, querying, and removing entity nodes.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] `register_node(&self, id: &EntityId, entity_type: EntityType, title: &str) -> Result<(), GraphError>` — upserts a node with label = entity_type, properties = {id, title}
- [ ] `remove_node(&self, id: &EntityId) -> Result<(), GraphError>` — deletes node and its edges
- [ ] `get_node(&self, id: &EntityId) -> Result<Option<NodeInfo>, GraphError>` — returns node metadata
- [ ] `has_node(&self, id: &EntityId) -> Result<bool, GraphError>`
- [ ] `NodeInfo` struct: id, entity_type, title
- [ ] Nodes use EntityId.to_string() as the graphqlite node ID
- [ ] Node label is the EntityType Display string
- [ ] `cargo build` succeeds

### Technical Approach
In `src/graph/nodes.rs`. Uses `graphqlite::Graph::upsert_node()` for register, `delete_node()` for remove, `get_node()`/`has_node()` for queries. Properties stored as HashMap<String, Value>.
