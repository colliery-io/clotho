## Objective

## Acceptance Criteria

- [ ] **GraphStore lifecycle**: Test open in_memory, verify stats show 0 nodes/0 edges
- [ ] **Node CRUD**: register node, has_node true, get_node returns correct info, remove node, has_node false
- [ ] **Node upsert**: register same node twice updates title, doesn't duplicate
- [ ] **Edge CRUD**: add edge between two nodes, has_edge true, get_edges_from returns it, remove edge, has_edge false
- [ ] **Edge with properties**: add edge with props, verify props are stored
- [ ] **Multiple edge types**: add BELONGS_TO and MENTIONS from same source, get_edges_by_type returns only matching type
- [ ] **Incoming edges**: add edges to a target, get_edges_to returns them
- [ ] **Relatable trait**: create entity, register as node, add edges, call relations() and verify results
- [ ] **Query helpers**: get_neighbors, get_related_by_type, get_incoming_by_type, get_entities_by_label
- [ ] **Raw Cypher**: execute a Cypher query and verify results
- [ ] **Edge removal cascades**: remove a node, verify its edges are also gone
- [ ] All tests pass with `cargo test`
