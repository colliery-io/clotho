# Batch relation creation tool

## Objective

Add a `clotho_batch_create_relations` MCP tool that accepts an array of relations in a single call, replacing the current pattern of one tool call per relation.

## Backlog Item Details

### Type
- [x] Feature - New functionality or enhancement

### Priority
- [x] P1 - High (important for user experience)

### Business Justification
- **User Value**: 297 individual `create_relation` calls were observed across 2 sessions. Linking an entity to 6 programs requires 6 separate tool calls. This is the single largest source of tool call volume.
- **Business Value**: Would cut ~50% of total MCP tool calls per session, reducing latency and token overhead.
- **Effort Estimate**: M

## Proposed Solution

Add `clotho_batch_create_relations` MCP tool accepting an array of `{source_id, relation_type, target_id}` objects. CLI equivalent: `clotho relate-batch` reading from stdin or args.

Should validate all inputs before creating any relations (atomic batch). Return a summary of created relations with IDs.
