# Workspace summary/dashboard tool

## Objective

Add a workspace summary tool that returns a high-level overview of the workspace in a single call, replacing the current pattern of enumerating every entity type individually.

## Backlog Item Details

### Type
- [x] Feature - New functionality or enhancement

### Priority
- [x] P2 - Medium (nice to have)

### Business Justification
- **User Value**: Session 1 starts with 14 consecutive `list_entities` calls (one per entity type) just to enumerate the workspace. There is no way to get a quick overview without iterating through every type.
- **Business Value**: Replace 14 `list_entities` calls with 1 summary call at session start. Faster session bootstrap, lower token cost.
- **Effort Estimate**: S

## Proposed Solution

Add `clotho_workspace_summary` MCP tool returning: entity counts by type, recent activity (last N created/updated entities), active/blocked task counts, unprocessed transcript count. CLI: `clotho summary` or `clotho status`.
